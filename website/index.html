The problem

Navigation was built for drivers, not pedestrians

MapQuest launched in 1996 for driving directions. Google Maps followed in 2005. Waze added crowdsourced traffic. The entire history of digital navigation has been about moving cars faster.

When these platforms added pedestrian routing, they used the same logic: shortest distance, fastest time. Walking got bolted onto a system designed for driving.

But drivers are in a metal box. Pedestrians are not. A driver doesn't care about streetlights or worry about being followed. A pedestrian does. The risk profile is completely different, and the navigation tools haven't caught up.

The solution

A safety layer any map can use

Safe Walk doesn't replace Google Maps. It adds what Google Maps is missing: a real-time safety score for every street segment in the city. Any app that uses Google Maps for walking directions can plug in Safe Walk's API and offer a "safest route" toggle alongside the fastest one.

Type your start and destination. Safe Walk calls the Google Routes API to get multiple walking route options.

Real data, not guesswork

Three layers of safety intelligence

Police incident data

Every reported assault, robbery, and break-and-enter from Toronto Police Open Data. Geocoded, weighted by severity and recency. An assault last month matters more than a bike theft two years ago.

Streetlight coverage

The location of every streetlight in Toronto from City Open Data. We compute a lighting density score for every street segment. Well-lit main roads score high. Dark side streets score low. After dark, this layer matters a lot.

Community reports

Users report harassment, poor lighting, and suspicious activity directly in the app. These reports feed back into safety scores in real time, capturing the things that never make it into police data.

For developers

One API call to make any route safer

Safe Walk works as a drop-in layer on top of Google Maps. Send us an origin and destination, get back routes ranked by safety.

POST /api/route
{
  "origin": "Robarts Library, Toronto",
  "destination": "100 Harbord St, Toronto"
}

// Response
{
  "fastest": {
    "distance": "1.2 km",
    "duration": "14 min",
    "safety_score": 0.42   // higher = more dangerous
  },
  "safest": {
    "distance": "1.6 km",
    "duration": "19 min",
    "safety_score": 0.18   // much safer
  }
}
